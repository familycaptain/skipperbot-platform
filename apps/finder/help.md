# Finder

One search box for everything in Skipper — find an item no matter which app it
lives in.

## Overview

Finder is a universal search launcher. Type once and it searches across the apps
that actually own your data — Goals, Documents, Recipes, Reminders, Schedules,
and more — then shows the matches grouped by app so you can jump straight to the
right place. Reach for it when you're not sure which app something is in. Finder
itself stores nothing; it just asks each app to search its own data.

## Screens

- **Search box.** Type any words; results stream in.
- **Results.** Grouped by app (e.g. a recipe, a document, a reminder), each with a
  link that opens it in its app.

## Example workflows

**Find something when you don't know where it is**
- *In the app:* open Finder, type "beach trip" → see the matching document,
  reminder, list, etc., and click through.
- *Through chat:* you can also just ask Skipper "find anything about the beach
  trip" — same idea, conversationally.

## Tips

- Use Finder for cross-app "where did I put that?" searches; use a specific app
  when you already know where the item lives.
- Results come from each app's own search, so they're as current as that app's data.

## Your data

Finder **stores nothing of its own** — there are no Finder records. It searches
the data held by the other apps (which is what's in Skipper's memory), so what
you can find here is exactly what those apps have saved.
