# Issues

Report bugs and request features for your Skipper — and follow them from "open"
all the way to a validated fix.

## Overview

Issues is your Skipper's own bug/feature tracker. Anyone in the household can
report a problem or request a capability; the developer (you, via the Evolve /
Claude Code workflow) picks it up, fixes it, and marks it for validation; then
the reporter confirms it works. It's the work queue that drives Skipper's
self-improvement, so it's a required, always-installed app.

## Screens

- **Issue list.** All issues with their type (bug/feature) and status; filter by
  status to see what's open, in progress, or awaiting your validation.
- **Issue detail.** The report (title, description, screenshots), the current
  **status**, and the **resolution** notes explaining what was done. You can edit
  the status and resolution here.
- **Report form.** File a new issue with a title, description, and optional screenshots.

## Status flow

`open → in progress → pending validation → fixed` (or `won't fix` / `duplicate`).
When something is **pending validation**, that's your cue to try it and confirm —
then it's marked **fixed**.

## Example workflows

**Report something**
- *In the app:* file a new issue — say what you expected vs. what happened, attach a screenshot.
- *Through chat:* "report a bug: the calendar shows the wrong day", "request a
  feature to export recipes to PDF".

**Track and validate**
- *In the app:* watch an issue move through the statuses; when it hits **pending
  validation**, test it and mark it **fixed** (or send it back if it's not right).
- *Through chat:* "what issues are pending validation?", "show my open issues".

## Tips

- Clear, specific reports get resolved faster — expected vs. actual, plus a screenshot.
- This app powers the self-improvement loop in the Evolve spec, which is why it
  can't be uninstalled.

## Your data

Issues, their status, resolution notes, and screenshots are **saved in the
database and pulled into Skipper's memory**, so you can ask "what did we fix last
week?" and Skipper knows. They stay within your household.
