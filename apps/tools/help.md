# Tools

An inspector for the MCP tools Skipper can use — see what capabilities are
available to the assistant.

## What you can do
- **Browse registered tools** and what each does.
- **Understand** what Skipper can act on (search, weather, calendar, app actions…).
- **Ask Skipper** — "what tools do you have?", or "can you look up the weather?".

## Tips
- This is a developer/power-user view; you mostly just ask Skipper in chat and it
  picks the right tool.
